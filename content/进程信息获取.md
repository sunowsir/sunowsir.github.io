* 知识准备：

  1. `ps` 该工具用来获取进程信息
  2. `awk`该工具用来处理数据

* 代码

  ```bash
  #!/bin/bash
  
  ps -aux | awk -v last_all=$(ps -aux | awk '
  BEGIN{
      num = 0;
  } 
  { 
      if ($3 > 50 || $4 > 50) { 
          if (num > 0) {
              printf("|");
          } 
          printf("%s", $2); 
          num++;
      } 
  }' && sleep 5) -v nowtime=$(date +"%Y-%m-%d__%H:%M:%S") '
  { 
      if ($2 ~ last_all && ($3 > 50 || $4 > 50)) {
          printf("%s %s %s %s %s %s\n", nowtime, $11, $2, $1, $3, $4);
      } 
  }'
  
  ```
