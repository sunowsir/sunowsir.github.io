linux三剑客之grep的使用
发布于 2019-09-02   更新于 2021-08-29 note

部分内容源自网络和grep帮助信息。

用法

常用参数选项说明

  • -o只显示与模式串匹配的内容。

  • -E 使用扩展正则表达式。

    栗子：

  • -v显示与模式串不匹配内容所在的整行内容。

  • -w只匹配整个单词，也就说只有独立的整个单词斗鱼模式串匹配才算。

  • -x只匹配整行，也就说只有整行都与模式串匹配才算。

  • -i忽略大小写。

  • -H显示每个符合样式字符串所在行所属的文件名。

  • -n显示行号。

  • -I忽略二进制文件。

更多用法请查看man手册或grep --help
